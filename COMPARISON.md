# Comparison

| Library                                                            | Description                                                                  |
| ------------------------------------------------------------------ | ---------------------------------------------------------------------------- |
| [`chess.js`](https://www.npmjs.com/package/chess.js)               | most popular JS chess library — move generation, validation, check/checkmate |
| [`chessops`](https://www.npmjs.com/package/chessops)               | TypeScript chess rules — supports variants (Crazyhouse, Antichess, etc.)     |
| [`js-chess-engine`](https://www.npmjs.com/package/js-chess-engine) | chess engine with configurable AI — no dependencies                          |
| [`chess.ts`](https://www.npmjs.com/package/chess.ts)               | TypeScript rewrite of chess.js                                               |
