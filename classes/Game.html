    Class Game

    A mutable chess game with legal move generation, undo/redo, and game-state detection.

    Example

    const game = new Game();
    game.move({ from: 'e2', to: 'e4' });
    game.move({ from: 'e7', to: 'e5' });
    game.moves('d1'); // legal queen moves
